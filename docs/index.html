Sobre

Meu nome é Davi Rodrigues Chiqueti. Sou natural de Florianópolis, Santa Catarina. Sou um profissional e estudante do ramo da Computação apaixonado por tecnologia, e constantemente em busca de explorar e contribuir para as inovações que moldam o mundo moderno.

Minha paixão pela computação e minha incessante curiosidade são os motores que me impulsionam a buscar novos aprendizados e evolução profissional.

Tecnologias

Sou apaixonado por tecnologia e utilizo uma stack robusta que me permite desenvolver soluções eficientes.
Aqui estão as principais ferramentas que utilizo em meu dia a dia profissional:

  • Python: Minha principal linguagem de programação
  • Docker: Utilizo Docker para garantir ambientes de desenvolvimento consistentes e escaláveis. Além de faciltiar a criação de aplicações.
  • Celery: Biblioteca para processamento distribuído de tarefas.
  • RabbitMQ: Serviço de mensageria implementado para gerenciar filas de tarefas.
  • PostgreSQL: Banco de dados relacional utilizado para armazenar dados estruturados.
  • Redis: Banco de dados em memória, utilizado para leitura e escrita otimizada em grande escala.
  • FastAPI: Framework para desenvolvimento de APIs Web de forma ágil e performática.
  • Git: Ferramenta de versionamento de código

Além disso, estou aprendendo algumas tecnologias novas durante a graduação e em meu trabalho
Sendo essas:

  • Apache Airflow: Para gerenciamento e monitoramento de workflows
  • C++: Linguagem de programação utilizada para o estudo de Estruturas de Dados
  • HTML & CSS: Linguagens de marcação e estilização usadas em aulas de Desenvolvimento Web

Vida Profissional e Acadêmica

Em 2023 ingressei no curso de bacharelado em Sistemas de Informação pela Universidade Federal de Santa Catarina (UFSC). Iniciando assim, minha jornada no ramo da Ciência da Computação.

Logo no início do curso, para complementar meu aprendizado nas aulas, me tornei membro da Pixel, empresa Júnior dos cursos de Sistemas de Informação e Ciência da Computação da UFSC. Além disso, iniciei um estágio em desenvolvimento Python voltado para a área de dados na Hooklab.

Atualmente, atuo como Desenvolvedor de Software Júnior na Hooklab, a mesma empresa em que realizei meu estágio. Na Hooklab, sou responsável pelo desenvolvimento e manutenção de aplicações de dados. Trabalho com fluxos de tarefas distribuídas, focando na extração eficiente de dados e na automação de processos, sempre colaborando em um ambiente ágil.
